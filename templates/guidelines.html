Usage Guidelines

How to Use the Parser

  1. Enter a complete Malayalam sentence in the text area on the home page.
  2. Click the "Parse Sentence" button to submit your text for analysis.
  3. The parsing results will appear below the input area, showing the dependency relationships between words.
  4. You can upload a text file containing multiple Malayalam sentences for batch processing.
  5. The system will parse each sentence and display the results in a structured table format.
  6. You can edit the dependency relations in the table if needed.
  7. After editing, click "Save & Download" to download a CoNLL-U file with the parsed sentences.

Best Practices

  • Use complete, grammatically correct sentences for best results.
  • Avoid very long or complex sentences (more than 25-30 words) as they may lead to less accurate parsing.
  • Ensure proper spacing between words.
  • The parser works best with standard Malayalam text without excessive code-mixing.

Input Limitations

The parser currently has the following limitations:

  • Maximum sentence length: 50 words
  • Supports only Malayalam script (with limited support for English words within Malayalam text)
  • May have reduced accuracy with dialectal variations

Understanding the Output

The parser output displays the dependency relationships between words in the sentence. Each word is connected to its head word with a labeled arc indicating the grammatical relationship. The output includes:

  • Word index (position in the sentence)
  • Word form (the actual word)
  • Part of speech (POS) tag
  • Head word index (the word it depends on)
  • Dependency relation label
