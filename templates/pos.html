Universal Dependencies

Universal Dependency is a framework for consistent annotation of grammatical relations across different languages. It provides a standardized way to represent syntactic structures using dependency trees. The UD framework includes a set of universal part-of-speech (POS) tags and dependency relations, making it useful for multilingual NLP tasks. The Universal Dependencies (UD) framework defines 17 universal part-of-speech (POS) tags applicable across languages, including Malayalam.

Malayalam POS Tags

POS Tag Description Example (Malayalam)
NOUN (നാമം) Common nouns പുസ്തകം (book), വിദ്യാർത്ഥി (student)
PROPN (വ്യക്തിനാമം) Proper nouns കേരളം (Kerala), രവി (Ravi)
VERB (ക്രിയ) Action or state verbs ഓടുന്നു (running), ലിഖിക്കുന്നു (writing)
ADJ (വിശേഷണം) Describes nouns വലിയ (big), നല്ല (good)
ADV (ക്രിയാവിശേഷണം) Modifies verbs, adjectives, or adverbs വേഗത്തിൽ (quickly), ഇന്ന് (today)
PRON (സർവ്വനാമം) Pronouns replacing nouns ഞാൻ (I), അവൻ (he)
DET (നിശ്ചയകാരണ) Determiners, articles, and quantifiers ഒരു (a), അത് (that)
ADP (ഉപസർഗം/ശേഷപ്രത്യയം) Prepositions and postpositions കൊണ്ട് (with), വരെ (until)
NUM (സംഖ്യ) Numbers (cardinal, ordinal) മൂന്ന് (three), ഒന്നാം (first)
CCONJ (സംയോഗ അവ്യയം) Coordinating conjunctions മറ്റുമിന്ന് (and), അല്ലെങ്കിൽ (or)
SCONJ (ആശ്രിതബന്ധക അവ്യയം) Subordinating conjunctions എന്നാൽ (if), കാരണം (because)
PART (അവ്യയം) Function words like negation or tense markers ഉം (also), മാത്രം (only)
AUX (സഹായക ക്രിയ) Auxiliary verbs ആണ് (is), ഉണ്ടു (exists)
X (അജ്ഞാതം) Foreign words, unclassified elements WiFi, OK, USB
PUNCT (വിശേഷധ്വനിക ചിഹ്നം) Punctuation marks . , ? ! " ( )
